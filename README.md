# WinTahoe
> The macOS Tahoe cursor pack for Windows.

![Preview](cover.png)

---

## Installation

1. **Clone this repository** or download it as a **ZIP** file and extract it.
2. Open the extracted **`cursors`** folder.
3. Right-click the **`install.inf`** file.
4. Select **Install** *(On Windows 11, you may need to click "Show more options" first)*.
5. Open the Windows Start Menu and search for **Mouse settings**.
6. Click on **Additional mouse options** (or "Additional mouse settings").
7. Navigate to the **Pointers** tab.
8. Select **WinTahoe** from the Scheme dropdown menu.
9. Click **Apply** and **OK**.

---

## A Note on Sizing

macOS cursors are inherently smaller than standard Windows cursors. If you wish to increase or decrease the size of your cursor, you must adjust the Windows scaling **before** applying this theme. 

To change the cursor size correctly:
1. Revert your cursor scheme back to the **Windows Default**.
2. Go to **Settings > Accessibility > Mouse pointer and touch**.
3. Adjust the **Size** slider to your preferred scale (you might want to set it slightly larger than your usual preference).
4. Once the desired size is set, follow the installation steps above to re-apply the **WinTahoe** scheme.

---

## Uninstallation

1. Open **Mouse settings > Additional mouse options > Pointers**.
2. Select **WinTahoe** from the Scheme dropdown menu.
3. Click the **Delete** button located next to the dropdown.
4. Click **OK** to confirm.
5. *(Optional)* To completely remove the files, navigate to `C:\Windows\Cursors\` and delete the `WinTahoe` folder.
